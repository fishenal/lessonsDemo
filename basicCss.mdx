### 一个 element 可能接受多个 css 规则

```html

```

```css
p {
  line-height: 50px;
}
.name {
  background: gray;
}
#my_name {
  color: red;
}
#your_name {
  color: yellow;
}
```

### 优先级

同级别下 inline style > id selector > class selector > tagname selector

```html
<span id="this_hello" class="common_span" style="color: blue;"
  >Hello world!</span
>
```

```css
#this_hello {
  color: red;
}

.common_span {
  color: orange;
}
span {
  color: yellow;
}
```

---

### 优先级

同类型下 限制的越精确优先级越高

```html
<div class="parent">
  <div class="child">
    <p class="my_name">Wendy</p>
  </div>
</div>
```

```css
.my_name {
  color: red;
}
.child .my_name {
  color: orange;
}
.parent .child .my_name {
  color: blue;
}
```

---

### 优先级

使用!important 提高特定属性的优先级

```css
.my_name {
  color: red !important;
}
.child .my_name {
  color: orange;
}
.parent .child .my_name {
  color: blue;
}
```

### 单位

```css
div {
  width: 500px; /* pixels*/
  height: 300pt; /* points*/

  width: 90%; /*percentage of parent*/
  padding: 5%; /*percentage of parent*/

  font-size: 1.2em; /* fontsize of parent*/
  line-height: 2.5rem; /* fontsize of root*/

  width: 1vw; /* viewport's width*/
  height: 1vh; /* viewport's height*/
}
```
